## 開発者向け情報
### 環境構築
#### 環境
- Java : `1.8.0`
- Gradle : `7.0.2`
- DB : `h2-database with PostgreSQL`
#### セットアップ
`gradlew setup`   
#### 環境変数
| 変数名 | 値| 説明 |
----|----|----
| LINE_BOT_CHANNEL_SECRET | 任意 | LINEAPI シークレット |
| LINE_BOT_CHANNEL_TOKEN | 任意 | LINEAPI トークン |
| SPRING_PROFILES_ACTIVE | dev or prod | dev を設定すればDBが/h2db/に生成される|
